# Reminisce - Vocabulary Learning App

A minimal, accessible vocabulary learning app built with SvelteKit.


## Tech Stack

- [SvelteKit](https://svelte.dev) - Framework
- [Supabase](https://supabase.com) - Authentication & Database
- [Tailwind CSS](https://tailwindcss.com) - Styling
- [Dictionary API](https://dictionaryapi.dev/) - Word definitions
- [Open Russian](https://openrussian.org/) - Russian vocabulary
